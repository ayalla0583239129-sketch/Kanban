# MédiatekFormation – Projet Symfony

Ce dépôt contient l’application **MédiatekFormation**, développée dans le cadre de l’atelier SLAM du BTS SIO option SLAM.  
L’application est une plateforme de gestion de playlists et de formations basée sur Symfony, Doctrine et Twig.:contentReference[oaicite:0]{index=0}

## 📌 Fonctionnalités ajoutées

### 🛠 Back Office
- Gestion des formations (ajout, modification, suppression)
- Gestion des playlists (ajout, modification, suppression)
- Gestion des catégories
- Sécurisation de l’accès à `/admin`

### 📊 Autres ajouts
